# Video-to-Audio-Extractor-Python-
A simple Python project that extracts audio from a video file and saves it as an MP3 file using the MoviePy library.

🚀 Project Description 

This project allows you to:
Load a video file
Extract its audio track
Save the audio as an .mp3 file

It is useful for:
Creating audio files from lectures or movies
Extracting podcasts from videos
Media processing projects

🛠️ Tech Stack
Python 🐍
MoviePy library 🎥
📦 Installation

Install the required library using:
pip install moviepy

▶️ How It Works
1. Import VideoFileClip
Loads video processing tools from MoviePy.

2. Load Video File
Reads the video file from your system.

3. Extract Audio
Gets the audio track from the video.

4. Save Audio File
Exports the audio as an MP3 file.

🎯 Features : 
🎥 Convert video to audio
🎧 Extract high-quality MP3 files
⚡ Simple and fast execution
📁 Works with most video formats supported by MoviePy
