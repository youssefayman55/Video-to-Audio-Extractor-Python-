# Video-to-Audio-Extractor-Python-
A simple Python project that extracts audio from a video file and saves it as an MP3 file using the MoviePy library.

🚀 Project Description 

This project allows you to:

1- Load a video file

2- Extract its audio track

3- Save the audio as an .mp3 file

It is useful for:

1- Creating audio files from lectures or movies

2- Extracting podcasts from videos

3- Media processing projects

🛠️ Tech Stack : 

1- Python 🐍

2- MoviePy library 🎥

📦 Installation:

Install the required library using:

pip install moviepy

▶️ How It Works:

1. Import VideoFileClip

Loads video processing tools from MoviePy.

2. Load Video File

Reads the video file from your system.

3. Extract Audio

Gets the audio track from the video.

4. Save Audio File

Exports the audio as an MP3 file.

🎯 Features : 

1-🎥 Convert video to audio

2- 🎧 Extract high-quality MP3 files

3-⚡ Simple and fast execution

4-📁 Works with most video formats supported by MoviePy
